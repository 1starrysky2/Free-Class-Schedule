# 无课表工具

这是一个用于**从课程表中自动提取无课时间**的小工具。  
你只需要上传教务系统导出的课表 Excel，工具就会按“星期 + 节次”计算每个时间段在哪些周是空闲的，方便安排自习、活动、面试或值班。

## 这个软件作用

- 读取 `.xlsx` / `.xls` / `.xlsm` 课表文件
- 识别课程周次信息（如 `1-16周`、`1-7周(单)`、`2-16周(双)` 等）
- 自动计算每个时间段的无课周次
- 结果按“星期 + 节次”输出（例如：`星期三 3-4节：5-8,10-16周空闲`）

## 适合谁用

- 想快速找“哪天哪节没课”的学生
- 需要排班/约时间但不想手动对照课表的人
- 想把“空闲时间计算”接入自己系统的开发者


## 基础使用方式

1. 启动程序（`python app.py` 或打包后的 exe）
2. 打开浏览器访问 `http://127.0.0.1:8001`
3. 上传课表文件并设置总周数
4. 获取自动生成的无课时间结果

## 打包
pip install pyinstaller
输入
pyinstaller build.spec --clean
等待即可打包为exe
